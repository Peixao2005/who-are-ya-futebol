# Who Are Ya? Futebol
Projeto de um web game inspirado no jogo Who Are Ya?, no estilo Wordle de futebol, focado em jogadores das 5 principais ligas europeias.
# Sobre o projeto
O objetivo do jogo é adivinhar um jogador misterioso a partir de pistas visuais e comparações de atributos. A cada palpite, o sistema mostra se a nacionalidade, o time, a posição, a idade e o número da camisa estão corretos ou não.
Tema
Tema: Futebol

O jogo utiliza atletas conhecidos das principais ligas da Europa, como Premier League, La Liga, Serie A, Bundesliga e Ligue 1.

# Proposta do jogo

# Objetivo
Criar uma experiência divertida e interativa de adivinhação de jogadores de futebol.

# O que o usuário encontra
- Campo de busca com autocompletar.
- Lista de jogadores disponíveis para chute.
- Feedback visual por atributos.
- Mensagem de vitória quando o jogador é descoberto.
- Botão para jogar de novo.

# Problema que o jogo resolve
O jogo transforma o conhecimento sobre futebol em uma experiência rápida e dinâmica, ideal para testar memória, atenção e familiaridade com jogadores famosos.
# Informações e serviços oferecidos
Jogo de adivinhação de jogadores.
Comparação de pistas por atributo.
Reinício de partida com novo jogador aleatório.

# Mobile First
O projeto segue o conceito Mobile First porque a maior parte do público acessa jogos pelo celular.
# Por que escolher esse modelo
O jogo precisa funcionar bem em telas pequenas, com botões fáceis de tocar e conteúdo organizado verticalmente.
# Como adaptar para celular
Campo de busca grande.
Menu e navegação simples.
Sugestões de jogadores em dropdown responsivo.
Cards de resultado organizados em coluna.
Botões com boa área de clique.
# Cuidados com o layout
Menu: mínimo e intuitivo.
Imagens: leves e otimizadas.
Textos: curtos e legíveis.
Tela: prioridade para o jogo e para as dicas.
