# Proposta de Site
# Objetivo
Criar um jogo interativo para fãs de futebol, com foco em adivinhação de jogadores famosos.
O que o usuário encontrará
Campo de busca com autocomplete.
Lista de jogadores para chute.
Comparação de atributos entre o palpite e o jogador secreto.
Tela de vitória e botão para reiniciar.
Problema que o site pretende resolver
O site organiza o desafio em uma experiência simples e divertida, transformando conhecimento de futebol em um jogo rápido no navegador.
Informações e serviços oferecidos
Jogo de adivinhação.
Feedback visual por atributos.
Reinício de partidas com novo sorteio.
